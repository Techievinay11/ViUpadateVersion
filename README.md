# Visuya – Make Better Money Decisions

A modern, fully functional **Personal Finance Website** built with pure HTML, CSS and JavaScript — ready to host on **GitHub Pages**.

## 🚀 Live Demo
> Host on GitHub Pages and your site is ready!

## ✨ Features

- **Hero Section** – Financial Overview widget with animated credit score gauge, sparkline chart, and savings progress bar
- **Top Categories** – Credit Cards, Banking, Loans, Credit Score, Investing, Insurance
- **6 Working Calculators** (Modal Popup):
  - 📊 EMI Calculator
  - 💰 Savings Calculator
  - 🏖️ Retirement Calculator
  - 📈 Compound Interest Calculator
  - ⭐ Credit Score Estimator
  - 🔍 Credit Card Finder
- **Best Credit Cards** – Dynamically rendered cards with ratings
- **Popular Comparisons** – Side-by-side card comparison grid
- **Credit Score Section** – Animated gauge with score ranges
- **Investing Section** – Stocks, Mutual Funds, ETFs, Crypto cards
- **Insurance Section** – Health, Life, Auto plans
- **Newsletter Signup** – With toast notification feedback
- **Live Search** – Real-time search with highlighted results
- **Mobile Responsive** – Full hamburger menu on mobile
- **Scroll Animations** – Cards animate in on scroll
- **Back-to-Top Button**
- **Active Nav Highlighting**

## 🌐 Deploy to GitHub Pages

1. Create a new GitHub repository
2. Upload all files (`index.html`, `style.css`, `app.js`, `README.md`)
3. Go to **Settings → Pages**
4. Set Source to `main` branch, root folder
5. Click **Save** — your site will be live at `https://yourusername.github.io/repo-name`
